HTML

HTML:
HyperText Markup Language
HTML is used to describe the struture of the website
HTML elements tells the browser how to display the content
HyperText means text within text,the text is used to link various webpages
Markup means Highligting the text by underlining or display it in different colors or both
Language means instruct the computer with syntax and rules

Tag:

the name of the content written in angular brackets

Element:

Element defined as start tag,content,end tag

The content enclosed between start tag and end tag

Self-closing Tags:

The tags which break the continuos content on the webpage

    Examples:
  • br tag
  • hr tag
  • Image tag

Inline Element:

the elements occupies only content space

    Examples:
  1. bold tag
  2. italic tag
  3. Strong tag
  4. emp tag etc

Block-level Elements:

Blocklevel elements always start in new line & occupies complete block where it start and where it ends

    Examples:
  1. heading tags
  2. paragraph tags
  3. blockquotes tag

Formattig tags

Formattig tags are used to desing the text in special types

    Examples:
  1. superscript tag
  2. subscript tag
  3. insert tag
  4. delete tag
  5. italic tag
  6. bold tag
  7. strong tag
  8. emphasis tag
  9. mark tag
