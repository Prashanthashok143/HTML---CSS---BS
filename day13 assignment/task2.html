css selectors

1.Element Selector:

Element Selectors are used to style the Elements based on Element name,id,class.

2.Universal Selector:

Universal Selector re used to style the Whole Webpage

3.Group Selector:

Group Elements are used to group mutliple elements and apply the same style for group elements

4.ID Selector:

ID Selector styles the element by using Unique Id

5.Class Selector:

Class Selector styles the element by using Class

6.Descendent Selector:

Descendent Selector matches all that are descendent of a specified element.

7.Child Selector:

Child Selector selects all elements that are the children of a specified element.
