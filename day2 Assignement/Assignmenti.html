1.What are Formatting Elements?
Formatting element are used to design the text in various formats.

Examples:

  • bold
  • italic
  • Small
  • strong

2.What are Semantic Elements?
Semantic Elements describes its meaning to both the browser and the developer

Examples:

  • heading tags
  • paragraph tags
  • strong tag
  • emp tag etc

3.What are Non-Semantic Elements?
Non-Semantic Elements doesn't describes about the content

Examples:

  • bold tag
  • italic tag etc

4.What is Inline Elements?
Inline Elements takes only Content space
5.What is mean by Block-Level Elements?
Block-Level Elements occupies complete block where it starts and where it ends
