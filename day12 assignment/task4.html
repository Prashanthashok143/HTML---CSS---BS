CSS

css element selector

the element selector selects html element based on the element name.

css id selector

The id selector uses the id attribute of an html element to specific element.

the id of an element is unique within a page,so the id selector is used to select one unique element.

if we want to specify the id,write (#) ,followed by the id of the element.

css class selector

the class selector selects html elements with specify class attribute.

if we want to specify the class,write (.) ,followed by the class of the element.
