1.What is HTML?

HTML stands for Hyper Text Markup Language

HTML describes the structure of a web pages

HTML tells the browser how to display the content

Hyper-Text: Means the text used to link various Webpages

Markup Language: means highlighting text enter by underlining or displaying it in different colors

2.What is TAG ?

3.What is Elements ?

The element is defined as content enclosed between open tag and closed tag

4.What is Self Closing tag or Open tag?

The self Closing tag is used for break the continues content in the webpage

Examples:
br:-This element creates a line break
hr:This element creates a horizontal line to divide content

Self closing tags are void(void means empty) elements

Formatting element are used to design the text in various formats

Semantic Elements describes its meaning to both the browser and the developer

Examples: heading tags, paragraph tags,Strong tag,emp tag etc

8.What are Non-Semantic Elements?

Non-Semantic Elements doesn't describes about the content

Examples: bold tag,italic tag etc
